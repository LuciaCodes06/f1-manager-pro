F1 Manager PRO
A desktop application for managing Formula 1 drivers using Python, Tkinter, and SQLite.

Features
* Add drivers with name, team, and points
* Automatically sorted leaderboard (by points)
* Update driver points (+10 system)
* Delete drivers
* Search drivers by name
* Persistent database using SQLite
* Clean GUI with Tkinter

Technologies Used
* Python
* Tkinter (GUI)
* SQLite (Database)

How to Run
1. Make sure Python is installed
2. Run the app:
```bash
python f1_pro.py
```
3. The database (`f1.db`) will be created automatically

How It Works
* Drivers are stored in a local SQLite database
* The app loads and sorts drivers by points
* Users can modify and search data in real time
* GUI updates dynamically using Tkinter

Future Improvements
